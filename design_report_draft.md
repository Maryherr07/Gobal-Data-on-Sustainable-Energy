# Design Report: Global Sustainable Energy Indicators Dashboard

## Introduction and analytical purpose

The dashboard was designed to support exploratory analysis of global sustainable energy indicators between 2000 and 2020. The overall theme is the relationship between energy access, clean energy transition, electricity generation, economic development and emissions. Instead of presenting the dataset as separate tables, the dashboard combines geographic, temporal, ranking and relationship-based visualisations in one interactive interface. This allows users to ask questions such as: Which countries have high electricity access but low clean cooking fuel access? How has low-carbon electricity changed over time? Which countries rely most heavily on fossil fuel electricity? Is economic development associated with higher energy consumption? The dashboard therefore communicates both progress and inequality in the global energy transition.

## Dashboard structure and visual hierarchy

The dashboard begins with a title, short explanation and KPI cards. These summarise average electricity access, clean fuels access, low-carbon electricity, renewable electricity generation and CO₂ emissions for the selected year. This creates a clear visual hierarchy: the most general information appears first, followed by more detailed views. The layout then uses a full-width map, paired two-column charts, and a final full-width correlation heatmap. This structure reduces cognitive load because users move from overview to comparison and then to deeper analytical relationships.

The dashboard contains eight visualisations. The first is a geographic map showing the selected metric by country. This is useful because sustainable energy indicators are strongly spatial, and a map helps reveal regional patterns. The second visualisation is a global average line chart over time, which communicates long-term progress in indicators such as electricity access, clean fuels access and low-carbon electricity. The third visualisation is a stacked area chart showing electricity generation from fossil fuels, nuclear and renewables. This reveals changes in electricity composition rather than only total generation.

The fourth visualisation is a country drill-down line chart. It allows selected countries to be compared over time using a chosen metric. The fifth visualisation is a ranked horizontal bar chart showing the top or bottom countries for a selected indicator. Ranking is useful when the analytical task is to identify leaders and laggards. The sixth visualisation is a scatter plot comparing GDP per capita with energy consumption per capita. This explores the relationship between economic development and energy demand, while colour represents low-carbon electricity. The seventh visualisation compares electricity access with clean fuels access, highlighting energy access inequality. The final visualisation is a correlation heatmap for the selected year, giving an overview of relationships between multiple indicators.

## Justification of chart choices and visual encoding

The chart types were selected to match the analytical purpose of each question. For time-based analysis, line charts and area charts are appropriate because position along a common horizontal time axis is easy to interpret. Position is one of the most accurate visual encodings, so it is suitable for comparing trends. The ranking chart uses horizontal bars because bar length is also effective for comparing numerical values. Horizontal orientation makes country names easier to read and reduces label clutter.

The map uses geographic position to support spatial reasoning and colour to encode the selected metric. This helps users quickly recognise regional differences. However, the map is not used as the only comparison tool because colour differences on maps can be less precise than position or length. Therefore, the ranked bar chart and line chart provide more accurate country-level comparisons. The scatter plots use x-position and y-position to show relationships between two quantitative variables. Colour adds a third variable, such as low-carbon electricity or renewable energy share, while bubble size represents estimated population. This makes the scatter plots multidimensional without requiring too many separate charts.

Colour is used mainly to encode meaning rather than decoration. Continuous colour scales support pre-attentive processing because high and low values can be detected quickly. Hover tooltips provide additional detail without overloading the initial view. This follows the principle of cognitive load reduction: the most important information is visible immediately, while exact values are available on demand.

## Interaction design

The main interactions are the year slider, country multiselect, metric selectors and top/bottom ranking control. These interactions are meaningful because they directly support exploratory analysis. The year slider updates the map, rankings, scatter plots, KPIs and correlation heatmap, allowing users to compare different points in time. The country multiselect updates the country trend line and electricity generation mix, creating a drill-down from global overview to selected national cases. The metric selectors allow users to reuse the same visual structure for different indicators, which improves consistency and learnability.

The interaction design follows the principles of affordance, feedback and consistency. Sliders and dropdowns clearly indicate what the user can change. The dashboard updates immediately after each selection, giving feedback that the interaction has worked. Controls are grouped in the sidebar so the main dashboard area remains focused on visual analysis. This separation helps reduce clutter and makes the interface easier to understand. Plotly hover interactions also allow users to inspect exact values without crowding the charts with permanent labels.

## Data preparation

Several data preparation steps were required before visualisation. First, long column names were renamed into shorter labels so that axis titles, legends and hover text would be readable. Second, the population density column contained formatted text, so it was converted into a numeric variable by removing commas and coercing invalid values to missing. Third, an estimated population variable was calculated using population density multiplied by land area. This was used for bubble sizing only, not as an official population measure. Fourth, total electricity generation was created by combining fossil, nuclear and renewable electricity generation. Missing values were not replaced with artificial values; instead, each chart drops missing values only for the variables used in that view.

A key data quality issue is that CO₂ emissions values are mostly missing for 2020. Because of this, the dashboard uses 2019 as the recommended default year for CO₂-based analysis. This is important because showing 2020 CO₂ totals without explanation could mislead users into thinking emissions dropped to zero. The dashboard includes a note to make this limitation transparent.

## Insights supported by the dashboard

The dashboard can reveal several important insights. First, energy access has generally improved over time, but the access scatter plot shows that electricity access and clean cooking fuel access do not always improve equally. Some countries may be close to universal electricity access while still having lower clean fuel access. Second, the energy generation mix shows whether renewable electricity is increasing compared with fossil fuel electricity. Third, the GDP and energy consumption scatter plot suggests that higher-income countries often consume more energy per person, but the colour encoding shows that high consumption does not always mean high low-carbon electricity. Finally, the ranking and map views show that global averages can hide strong country-level differences. Overall, the dashboard supports a coherent narrative: sustainable energy progress is visible globally, but it is uneven across countries and across dimensions of the energy system.
